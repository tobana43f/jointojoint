### ゲーム概要

「**繋ぐ**」をテーマにした作品です。

プレイヤーは、キャラクターのコアと呼ばれる球体を操作し、
他のキャラクターや、ステージ上のオブジェクトに繋ぐことで、
ステージを攻略するアクションパズルゲームとなっています。
